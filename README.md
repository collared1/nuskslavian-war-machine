## Install Instructions
1. download the file `bot.py`
2. put it into a folder in Documents
3. add to the folder an image called "NukeBotImage.png" (capitalization matters)
4. in the file, edit all of the config options, and add a bot token from the [discord developer portal](https://discord.com/developers/applications) into the tokens section, following instructions in the code.
5. go to Command Prompt or Terminal.
6. run `cd "C:Users/YOUR WINDOWS NAME/Documents/YOUR FOLDER NAME"` if you are on windows, and `cd "~/Documents/YOUR FOLDER NAME"` if you are on mac.
7. run `pip install discord.py`
8. run `python bot.py`
9. add the bot to a server using the invite link the bot generates
10. run `!nuke` in the server.
11. watch the nukebot do its thing
